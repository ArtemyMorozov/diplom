1️. Зарегистрируй своё приложение в Яндексе
Перейди на OAuth Яндекса.

Введи название приложения (например, "My Smart Home App").

В разделе "Доступы" выбери "Яндекс Умный дом API" (иногда может называться IoT API).

Нажми "Создать".

Скопируй Client ID (он понадобится для получения токена).

2️. Получи OAuth-токен с правильным scope
Теперь, когда у тебя есть свой Client ID, можно запросить OAuth-токен.

Открой браузер и вставь ссылку (заменив ВАШ_CLIENT_ID на свой ID):
https://oauth.yandex.ru/authorize?response_type=token&client_id=ВАШ_CLIENT_ID&scope=cloud_api:iot
